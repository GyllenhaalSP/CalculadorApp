# CalculadorApp

## Descripción
Este es un proyecto de prueba para ver cómo funcionan C# y .NET 8 con interfaz de usuario.
Es una calculadora simple que permite realizar operaciones matemáticas básicas y algunas funciones avanzadas.
Se ha usado WinForms para la interfaz de usuario.

## Características
- Operaciones básicas: suma, resta, multiplicación, división y módulo.
- Funciones avanzadas: cálculo de porcentajes, logaritmo en base 10, potencias y raíces cuadradas.
